## Book

O book é como o cartão de visita de um modelo. Trata-se de uma coleção de fotos profissionais que destacam as melhores características e versatilidade do modelo. Para os modelos mirins, é essencial ter um book que mostre sua personalidade, expressões e habilidades de posar para as câmeras.

## Casting

Os castings são como entrevistas para modelos. É onde os candidatos são avaliados por clientes ou agências para diferentes trabalhos, como desfiles de moda, comerciais ou editoriais. Para se destacar em um casting, os modelos mirins devem mostrar confiança, versatilidade e uma atitude profissional.

## Fitting

No fitting, os modelos experimentam roupas para garantir que elas se ajustem perfeitamente antes de um trabalho. Para os modelos mirins, é importante estar confortável e seguir as instruções dos estilistas para garantir um ajuste perfeito.

## Agência de Modelos

Uma agência de modelos é como uma equipe de apoio para modelos. Elas representam os modelos, ajudam a encontrar oportunidades de trabalho, negociam contratos e fornecem suporte profissional para o desenvolvimento da carreira. Encontrar uma agência de modelos respeitável é fundamental para os modelos mirins e suas famílias.

## Desfile de Moda

Os desfiles de moda são eventos emocionantes onde os modelos têm a oportunidade de mostrar roupas e acessórios de designers na passarela. Para os modelos mirins, desfilar é uma ótima maneira de ganhar experiência e visibilidade na indústria da moda.

## Editorial

Editoriais são sessões de fotos organizadas para publicações de moda, como revistas. Essas fotos são cuidadosamente planejadas para contar uma história ou transmitir um conceito. Participar de editoriais pode ser uma experiência empolgante e uma oportunidade para os modelos mirins trabalharem com fotógrafos e estilistas talentosos.

## Test Shoot

Um test shoot é uma sessão de fotos realizada para criar novas imagens para o portfólio do modelo ou colaborar com fotógrafos iniciantes. Essas sessões são uma ótima maneira para os modelos mirins experimentarem diferentes estilos e técnicas de fotografar Casting Director

O casting director é a pessoa responsável por selecionar modelos para trabalhos específicos. Eles procuram modelos que se encaixem no perfil desejado pelo cliente e que tenham a habilidade de trazer vida à visão do designer ou diretor de arte.

## Call Time

O call time é o horário em que um modelo deve estar pronto para iniciar um trabalho ou evento. É importante que os modelos mirins cheguem no horário marcado, preparados e prontos para trabalhar.

## Runway Walk

O runway walk é a técnica de caminhar na passarela com confiança e elegância durante um desfile de moda. Para os modelos mirins, dominar o runway walk é essencial para se destacar e impressionar os espectadores e clientes.
